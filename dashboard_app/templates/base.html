Dashboard Terminology Guide
Overall Phishing Funnel

These charts provide a high-level summary of user actions across all of your phishing campaigns combined.

  • Sent: The total number of phishing emails sent.
  • Opened: The number of emails where the tracking image was loaded. This metric can be unreliable as many email clients block images by default.
  • Clicked: The number of users who clicked the phishing link in the email. This is a primary indicator of user susceptibility.
  • Submitted: The number of users who entered data into the landing page. This is the most critical failure event.
Fail Rate

This metric measures the risk level of your organization. It is calculated with the following formula:

(Total Clicks + Total Submissions) / Total Emails Sent * 100%

A user who only opens the email is not counted in the fail rate.
